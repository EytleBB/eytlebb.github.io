# 网站维护手册

---

## 一、日志（Patch Log）

**两步添加一篇日志：**

**第一步** — `js/main.js`，在 `patchlog` 数组里加日期：
```js
patchlog: [
  '2026-04-10',
],
```

**第二步** — 在 `logs/` 目录下新建 `2026-04-10.txt`，直接写正文：
```
今天做了什么……

空行就是换段，想写多长写多长。
```

> 日期格式必须是 `YYYY-MM-DD`，月和日补零（`04` 不能写 `4`）

---

## 二、画廊（Gallery）

**第一步** — 把图片放进 `images/gallery/`

**第二步** — `js/main.js`，在 `gallery` 数组里加一行：
```js
gallery: [
  { src: 'images/gallery/photo.jpg', title: '图片说明' },
],
```

---

## 三、项目（Projects）

`js/main.js` → `projects` 数组

**无子项目**（点击直接跳 GitHub）：
```js
{ id: 'csai', name: 'CSAI', github: 'https://github.com/EytleBB/CSAI', sub: [] },
```

**有子项目**（点击后在第4列展开子列表）：
```js
{ id: 'myproj', name: 'MyProj', github: 'https://github.com/...', sub: [
  { name: '子项目A', github: 'https://github.com/.../A' },
  { name: '子项目B', github: 'https://github.com/.../B' },
]},
```

---

## 四、工具（Tools）

`js/main.js` → `tools` 数组：
```js
{ name: '工具名', nameEn: 'Tool Name', url: './工具页面.html', external: false },
```

> `external: false` → 在当前页打开  
> `external: true` → 新标签打开外部网址

---

## 五、下载（Downloads）

`js/main.js` → `downloads` 数组：
```js
{ name: '文件名', nameEn: 'File Name', meta: '说明', icon: '📦', url: './files/xxx.zip' },
```

---

## 六、概况（About）

`js/main.js` → `about`：
```js
about: {
  email: '你的邮箱',
  github: 'https://github.com/你的用户名'
},
```

---

## 七、图片

| 用途 | 路径 |
|---|---|
| 第1列圆形 Logo | `images/logo.png` |
| 画廊图片 | `images/gallery/任意文件名.jpg` |
| Patch Log 背景 | `images/patchlog-bg.jpg` |

放好后：

- **Logo** — 打开 `index.html`，找到 `.brand-logo-circle`，把注释替换为：
  ```html
  <img src="images/logo.png" alt="logo" />
  ```

- **Patch Log 背景** — 打开 `css/style.css`，找到 `.patchlog-bg`，取消这行注释：
  ```css
  /* background-image: url('../images/patchlog-bg.jpg'); */
  ```

---

## 八、日志字体

1. 把字体文件（`.woff2` / `.woff`）放进 `fonts/`
2. 打开 `css/style.css` 顶部，取消 `@font-face` 的注释，填上文件名
3. 找到 `.plog-body`，取消 `font-family` 那行的注释

---

## 文件结构速查

```
eyt_web/
├── index.html          # 主页结构（一般不动）
├── js/
│   └── main.js         # ★ 所有数据在这里改
├── css/
│   └── style.css       # 样式（改背景图/字体时动）
├── logs/
│   └── YYYY-MM-DD.txt  # ★ 每篇日志一个文件
├── images/
│   ├── logo.png
│   ├── patchlog-bg.jpg
│   └── gallery/
├── fonts/              # 上传字体放这里
└── files/              # 下载资源放这里
```
